初览
============================

要快速了解FinRL是什么以及它如何工作，您可以浏览Stock_NeurIPS2018系列，包括我们examples目录中的*Stock_NeurIPS2018_Data.ipynb*、*Stock_NeurIPS2018_Train.ipynb*、*Stock_NeurIPS2018_Backtest.ipynb*（https://github.com/AI4Finance-Foundation/FinRL/tree/master/examples）

这是我们如何从零开始使用深度强化学习进行股票交易的方法。


.. tip::

    在`Google Colab`_上逐步运行代码。

    .. _Google Colab: https://colab.research.google.com/github/AI4Finance-Foundation/FinRL-Tutorials/blob/master/1-Introduction/Stock_NeurIPS2018_SB3.ipynb

该笔记本及以下结果基于我们的论文*Practical deep reinforcement learning approach for stock trading* Xiong, Zhuoran, Xiao-Yang Liu, Shan Zhong, Hongyang Yang, and Anwar Walid. "Practical deep reinforcement learning approach for stock trading." arXiv preprint arXiv:1811.07522 (2018)。
